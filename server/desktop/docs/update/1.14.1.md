## ✅ Lo nuevo
 
### Filtro GGN — Módulo de Lotes
El filtro GGN en **Inventarios → Lotes** ahora tiene tres opciones: **Todos**, **GGN** y **Sin GGN**. Antes solo era posible activar o desactivar el filtro; ahora se puede filtrar específicamente la fruta sin certificación GGN.
 
### Lista de Empaque — Códigos ENF
El documento de Lista de Empaque (Packing List Report) fue actualizado a la **versión 04** (PC-CAL-FOR-05). Los cambios principales:
 
- Se agregó el **código ENF del lote** como columna visible en el reporte.
- Se agregó la **fecha de vencimiento GGN del predio** al final del documento.
- Todos los clientes reciben ahora el **mismo formato de 11 columnas**, independientemente de si son CoC o no.
### Nuevo módulo: Historial de Listas de Empaque
Nuevo módulo para consultar y descargar el historial de listas de empaque de contenedores exportados. Incluye:
 
- Vista de detalle con toggle entre **formato local** y **formato cliente**.
- Toggle entre **Lista de Empaque** y **Reporte de Predios**.
- Consulta paginada (50 contenedores por página).

## 🔧 Correcciones
 
Se corrigieron 4 errores encontrados durante el desarrollo:
 
- **Filtro GGN no retornaba resultados** — el valor enviado desde la pantalla era texto pero la base de datos lo espera como verdadero/falso. Se corrigió la conversión antes de consultar.
- **Error al guardar registros en Lista de Empaque** — varias operaciones sobre la base de datos corrían en paralelo y generaban conflicto. Se corrigió para que se ejecuten en secuencia.
- **Error al iniciar el servidor** — los esquemas de guardado de la base de datos no eran compatibles con la versión actual de Mongoose. Se actualizaron todos los afectados.
- **Error al abrir el historial** — ocurría cuando un predio asociado a un pallet había sido eliminado. Se corrigió para mostrar "N/A" en esos campos en lugar de fallar.
